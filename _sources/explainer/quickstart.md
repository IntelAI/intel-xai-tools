# Quick Start


## Build and Install

### Requirements

- Linux system (or WSL2 on Windows)
- git
- python >= 3.9

### Steps

1. apt-get install build-essential python3-dev
1. git clone https://github.com/intel-innersource/frameworks.ai.explainable-ai-tools
1. cd frameworks.ai.explainable-ai-tools
1. pip install -U -e .
